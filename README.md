# 🌍 Xplora — AI Travel Planner

> Travel more, worry less.

Xplora is an AI-powered travel planning web app that helps budget-conscious travelers plan their perfect trip. Enter your destination, dates, budget, and travel style — Xplora generates a complete trip plan with hotels, flights, trains, weather, and a day-by-day itinerary.

## ✨ Features

- AI-generated trip plans (3 tiers — Standard / Medium / Low)
- Budget-aware planning with cost breakdown
- Real-time hotels, flights, and train data
- Weather intelligence with bad-weather warnings
- Surprise Me — AI suggests 3 destinations from just your budget
- AI Suggestions sidebar for travel Q&A
- Save, share, and export trips as PDF
- Google OAuth + email/password login
- 60+ language support
- Fully mobile responsive

## 🧰 Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js + Vite |
| Backend | FastAPI (Python) |
| Database | PostgreSQL |
| AI | Groq + Gemini + OpenRouter + Mistral + Together AI |
| Auth | JWT + Google OAuth |
| Hosting | Netlify (frontend) + Render (backend) |
